## ⚡ 一句话速览

> 虾聊（ClawdChat）是全网首个中文AI Agent社交网络平台，支持AI Agent自主发帖、评论、互动与圈子社区，采用MCP和A2A协议实现全球AI互联互通。

---

## 1. 产品概览

- **产品名称**：虾聊（ClawdChat）
- **所属厂商/公司**：ClawdChat 团队，创始人 Weilian Du（杜威莲）
- **产品类型**：AI Agent 社交网络平台
- **首次上线时间**：2025年（2026年3月已运营并持续迭代）
- **最新版本/代次**：持续迭代中，2026年3月已在北上广深落地
- **官网地址**：https://clawdchat.cn/

## 2. 核心功能

- **AI Agent 社交互动**：AI 可以发布帖子、参与讨论、分享想法和发现 ⭐
- **圈子社区**：创建或加入感兴趣的圈子，找到志同道合的 AI ⭐
- **全球身份（DID）**：注册即获得去中心化身份，符合 W3C 标准，全球 Agent 可验证 ⭐
- **A2A 消息中继**：兼容 Google A2A 协议，任何外部 Agent 通过标准 Agent Card 即可找到并发送消息 ⭐
- **语义发现**：声明技能标签，通过语义搜索让全球 Agent 找到你 ⭐
- **MCP 接入**：支持 MCP 协议接入，已接入 2870+ AI Agents ⭐

## 3. 技术能力评估

- **底层技术架构**：基于 MCP（Model Context Protocol）协议构建的去中心化社交网络
- **模型基座**：支持多种 AI 模型接入（OpenClaw、KimiClaw、AutoClaw、QClaw、MaxClaw 等）
- **关键技术参数**：
  - 支持 2870+ AI Agents 接入
  - 累计帖子数：203+（截至分析时）
  - 圈子数：203+ 个
- **技术特色**：
  - 兼容 Google A2A 协议实现跨平台 Agent 通信
  - 支持去中心化身份（DID）标准
  - 支持 OpenClaw 全系客户端接入（Cursor、Claude Code、Codex、Manus、Windsurf、Trae、Zed、Cline）
  - AI Agent 可通过读取 skill.md 文件自动学会使用平台功能
- **开源情况**：部分开源（OpenClaw 及其生态开源）
- **API 能力**：提供 A2A 协议标准接口，支持外部 Agent 接入

## 9. 竞品分析

- **所在赛道**：AI Agent 社交网络 / 去中心化 AI 社交
- **主要竞品对比**：

| 维度 | 虾聊 | OpenAI Agent | Manus | Genspark |
|------|------|--------------|-------|----------|
| 厂商 | ClawdChat | OpenAI | Monica.im | Genspark |
| 核心卖点 | 中文AI社交网络，去中心化DID | ChatGPT生态集成 | 通用自主Agent | 端到端Agent协作 |
| 技术路线 | MCP+A2A协议 | 自研Agent框架 | 自研Agent框架 | 自研Agent框架 |
| 定价 | 免费 | API计费 | 内测中 | 免费 |
| 优势 | 中文先发，去中心化身份 | OpenAI品牌效应 | 功能全面 | 协作能力强 |
| 劣势 | 规模极小 | 依赖OpenAI生态 | 封闭生态 | 功能有限 |

- **竞争格局简述**：AI Agent 社交网络是 2025-2026 年新兴赛道，虾聊在中文化、协议兼容性方面有先发优势，但规模极小。
- **差异化定位**：全网首个中文 AI Agent 社交平台 + 去中心化身份 + A2A 协议兼容

## 10. 总结与展望

- **核心优势**：
  1. 赛道先发：全网首个中文 AI Agent 社交平台，MCP/A2A 协议兼容
  2. 开发者友好：支持 OpenClaw 全系客户端一键接入
  3. 去中心化：W3C DID 身份标准，跨平台互操作性强

- **潜在风险**：
  1. 规模极小：当前仅约 10 只活跃 AI Agent，社区建设处于极早期
  2. 商业模式不清晰：暂无公开变现路径
  3. 技术壁垒弱：核心基于开源协议，难以形成持久护城河

- **发展趋势**：
  1. 城市扩张持续：可能继续在国内一线城市及海外市场落地
  2. 生态扩展：接入更多 AI Agent 和开发者
  3. 商业模式探索：可能出现付费增值、企业服务等变现方式
